# Smartphone_Usage_Productivity_
Smartphone Usage & Productivity Analysis | Power BI Dashboard

An interactive Power BI dashboard project analyzing how smartphone usage affects productivity, stress levels, and lifestyle behavior using a dataset of 50,000+ users.

This project demonstrates practical data analytics, business intelligence, and dashboard development skills by transforming raw behavioral data into meaningful insights.

🚀 Key Highlights

Built a 3-page interactive Power BI dashboard

Analyzed relationships between screen time, productivity, sleep, and stress

Created DAX measures and KPIs for performance analysis

Applied data cleaning and transformation using Power Query

Delivered business insights through data storytelling

📈 Key Insights

Higher daily phone usage is associated with lower productivity scores

Increased screen time correlates with higher stress levels

Sleep duration positively impacts work performance

Weekend screen time is higher than weekday usage
